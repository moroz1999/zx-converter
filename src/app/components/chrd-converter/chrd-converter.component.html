Client-side converter from PC images to ZX Spectrum's chr$

https://github.com/moroz1999/chrd-converter - sources at Github

How to use

  1. Prepare PNG image in some PC editor. Image width/height should be a multiple of 8
  2. Load it
  3. Convert it to see the preview
  4. Download result in chr$ format

Notes: Only standard mode (type 9) is supported.
