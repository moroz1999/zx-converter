Client-side converter from PC images to ZX Spectrum's chr$

Simple to use image converter to chr$ format.
https://github.com/moroz1999/chrd-converter - sources at Github

How to use

  1. Prepare PNG image in some PC editor
  2. Load it
  3. Convert it to see the preview
  4. Download file

This is a beta version. Only standard mode (type 9) is supported currently. There are no format/size checks, errors, anything - use at your own risk.
